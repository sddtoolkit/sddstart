# analyze change impact skill

## 元信息
- 版本：v1.0
- 生效日期：2026-02-28
- 最后更新：2026-03-02
- 变更说明：新增变更影响范围分析方法。

## 能力描述
- 识别变更对需求、设计、任务、测试与发布范围的影响。

## 使用场景
- 任务规划阶段的影响评估。
- 发布前的回归范围评估与风险判断。

## 输入格式
- 变更说明、受影响模块、追溯矩阵、测试结果。

## 输出格式
- 影响范围清单（功能、模块、文档、测试）。
- 风险等级与回归建议。

## 执行步骤
1. 识别变更触发点与受影响链路。
2. 评估受影响需求与设计条目。
3. 评估任务与测试回归范围。
4. 输出优先级与缓解建议。

## 约束条件
- 不承担最终发布审批与技术实现决策。
