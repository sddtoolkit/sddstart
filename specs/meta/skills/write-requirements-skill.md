# write requirements skill

## 元信息
- 版本：v1.0
- 生效日期：2026-02-28
- 最后更新：2026-03-02
- 变更说明：模板能力基线。


## 能力描述
- 依据标准模板产出结构化需求文档，覆盖功能、非功能与验收标准。

## 使用场景
- 新需求立项、范围重定义或版本需求冻结前。

## 输入格式
- 业务目标、用户场景、约束、合规要求、风险信息。

## 输出格式
- 完整需求文档。
- 验收标准清单。
- 与设计/任务/测试的追踪入口。

## 执行步骤
1. 组织目标与范围。
2. 编写 FR/NFR 与数据合规要求。
3. 定义可测验收标准。
4. 标注风险与开放问题。

## 约束条件
- 必须可测试、可追溯。
- 外部事实必须登记来源。
