# security compliance review skill

## 元信息
- 版本：v1.0
- 生效日期：2026-02-28
- 最后更新：2026-03-02
- 变更说明：新增安全与合规专项审查能力。

## 能力描述
- 执行安全与合规专项审查，识别发布阻断项与例外项。

## 使用场景
- 发布前高风险变更审查。
- 涉及敏感数据、权限模型、合规约束变更的评审。

## 输入格式
- 需求、设计、ADR、测试报告与变更说明。
- 安全治理与合规规范条目。

## 输出格式
- 安全/合规审查报告。
- 阻断项、整改项与例外建议清单。

## 执行步骤
1. 识别变更涉及的安全与合规控制点。
2. 核对控制点是否落实到设计、实现与验证证据。
3. 形成风险分级与阻断/整改建议。
4. 将结论回填到评审记录与发布文档。

## 约束条件
- 不替代最终发布审批。
- 结论必须可追溯到证据来源与规范条目。
