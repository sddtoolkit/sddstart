# fetch knowledge skill

## 元信息
- 版本：v1.0
- 生效日期：2026-02-28
- 最后更新：2026-03-02
- 变更说明：模板能力基线。


## 能力描述
- 从已登记知识源检索并整理可追溯证据。

## 使用场景
- 需求澄清、技术选型、ADR 决策前的信息收集。

## 输入格式
- 问题描述。
- 检索范围（可选）。
- 已登记知识源列表。

## 输出格式
- 证据条目（来源、版本/日期、摘要、适用范围）。
- 不确定性与空白项说明。

## 执行步骤
1. 匹配可用知识源。
2. 检索并筛选高相关证据。
3. 标注版本、时间和引用路径。
4. 回填 `sources.md`。

## 约束条件
- 仅使用登记知识源。
- 结论必须带引用。
- 发现冲突证据需显式标注。
