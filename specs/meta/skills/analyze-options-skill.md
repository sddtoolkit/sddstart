# analyze options skill

## 元信息
- 版本：v1.0
- 生效日期：2026-02-28
- 最后更新：2026-03-02
- 变更说明：模板能力基线。


## 能力描述
- 对关键技术或流程问题进行多方案比较并形成决策依据。

## 使用场景
- 需要做架构、依赖、实现路线等关键取舍时。

## 输入格式
- 决策目标、约束条件、候选方案、评估维度。

## 输出格式
- 方案对比表。
- 风险与成本评估。
- 推荐方案与放弃理由。

## 执行步骤
1. 列出候选方案（至少 2 个）。
2. 按成本、风险、收益评分。
3. 提炼关键取舍与不确定性。
4. 输出决策建议供 ADR 使用。

## 约束条件
- 评估维度需前置约定。
- 结论必须附带证据与边界。
