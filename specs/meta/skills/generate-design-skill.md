# generate design skill

## 元信息
- 版本：v1.1
- 生效日期：2026-02-28
- 最后更新：2026-03-02
- 变更说明：补充前置 Skill 输出输入约定。


## 能力描述
- 将需求与 ADR 转化为可实施的架构与详细设计。

## 使用场景
- 需求冻结后、实现任务拆分前。

## 输入格式
- 需求文档、相关 ADR、技术约束。
- 前置 Skill 输出（可选）：`draft-adr-skill` 形成的 ADR 决策结论。

## 输出格式
- 架构设计文档。
- 接口与数据设计草案。
- 风险与权衡说明。

## 执行步骤
1. 划分模块边界与职责。
2. 定义关键流程与接口契约。
3. 明确安全、性能、可观测性策略。
4. 回填追踪关系。

## 约束条件
- 设计必须可映射到任务与测试。
- 关键路径必须给出性能目标与回滚策略。
