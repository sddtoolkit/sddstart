# estimate effort skill

## 元信息
- 版本：v1.0
- 生效日期：2026-02-28
- 最后更新：2026-03-02
- 变更说明：新增任务工时估算与排期评估能力。

## 能力描述
- 对任务分解项进行工作量、依赖与风险评估，形成可执行排期与缓冲建议。

## 使用场景
- 任务计划编制与迭代排期调整时。
- 需求变更后评估交付影响时。

## 输入格式
- 任务列表与依赖关系。
- 资源约束、技能分布、里程碑目标。
- 风险项与历史执行数据（如有）。

## 输出格式
- 任务级估算（人天/复杂度）。
- 关键路径与缓冲建议。
- 风险对排期影响说明。

## 执行步骤
1. 校验任务粒度与依赖完整性。
2. 估算任务规模并标注不确定性。
3. 识别关键路径与瓶颈资源。
4. 输出基线排期与风险缓冲建议。

## 约束条件
- 本 Skill 提供估算建议，不替代 Owner 的优先级与资源决策。
- 估算结论必须可解释并关联任务依据。
