# generate changelog release skill

## 元信息
- 版本：v1.0
- 生效日期：2026-02-28
- 最后更新：2026-03-02
- 变更说明：模板能力基线。


## 能力描述
- 将发布变更、风险与验证结果结构化回填到发布与变更文档。

## 使用场景
- 版本发布前后、热修复发布、回滚发布记录。

## 输入格式
- 发布版本、变更清单、风险评估、验证结果。

## 输出格式
- 更新后的发布记录。
- 更新后的变更日志条目。
- 关联追踪项清单。

## 执行步骤
1. 汇总版本变更与影响范围。
2. 记录风险、回滚和验证结果。
3. 回填发布计划与 CHANGELOG。
4. 同步关联需求/任务/测试信息。

## 执行方式
- 发布文档创建：`python3 specs/meta/tools/sddtool.py create-release <version>`
- 变更日志检查：`python3 specs/meta/tools/sddtool.py check-changelog`

## 约束条件
- 关键变更必须记录风险与回滚。
- 变更记录需可追溯到任务与验证证据。
